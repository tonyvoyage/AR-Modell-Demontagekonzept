# AR-Modell Demontagekonzept

Dieses Projekt enthält einen webbasierten AR-Viewer
für das im Rahmen eines Technikerprojektes erstellte Demontagekonzept.
Der Auftraggeber ist cunova.

## Funktionen

- 3D-Ansichten
- AR-Darstellung
- Interaktive Navigation
